Potentiometer - Programming

In this page we'll see how to take an analog input from a potentiometer and we will try understanding the code that makes everything work as we go.

First let's start with the wiring

Here is what we will need:
Of course the esp32
One green LED, one yellow LED and one red LED
A potentiometer
Three 220 Ohm resistors
A breadboard
And enough cables!

First we'll wire the LED'S while keeping in mind that the long side is the one that will connect to a GPIO and the short side willl be a ressistor

Then we'll wire the potentiometer as follows:

Here is how the full wiring should look like, make sure to connect the ground pin on the esp32 to the minus bands on the bread board
so the potentiometer and all the resistors can go there

when connecting the potentiometer to the esp32 make sure to connect it to one of the GPIO pins that has the ADC2 function.
here is a picture showing what each pin has

With this done we can move onto the programming!

Let's start by making a new sketch you should get something like this:

void setup() { // put your setup code here, to run once: } void loop() { // put your main code here, to run repeatedly: }

Any code written within void setup will run once at the very start of the program
and any code written within loop will run, like the name says on loop
You can also use the space above void setup to declare global variables like this:

const int rLedPin = 18; const int yLedPin = 19; const int gLedPin = 21; const int potPin = 4; int potVal = 0; void setup() { }

const meaning constant defines a veriable that does not change throughout the entire program
int defines a variable that consists of a whole number

rLedPin, yLedPin and gLedPin declare the GPIO port that you used for each led, so when writing your code for this make sure to update these to be accurate to your real wiring!

potPin is similar and defines the pin of the potentiometer

potVal is a veriable that we will use later on that will help us store the current value of the Potentiometer, which is why this isn't a constant as it will change

now, here is what we will be writing in the setup:

void setup() { Serial.begin(115200); pinMode(rLedPin, OUTPUT); pinMode(yLedPin, OUTPUT); pinMode(gLedPin, OUTPUT); }

Serial.begin(number) will start the serial console for debugging that we have used in previous tutorials, the number sets the baud rate that will be used, the one we typically use being 115200

pinMode(pin, OUTPUT)
declares wether a selected pin is an INPUT or an OUTPUT, we use the constants we declared earlier instead of the pin number so that we make the code more understandable at a glance and easily adaptable

for example if we were using a pushbutton we would declare
pinMode(buttonPin, INPUT)
instead, this is however not required for the analog potentiometer we will be using

also very Important to remember, ; marks the end of a line!

now, let's begin programming our loop program:

void loop() { potVal = analogRead(potPin); Serial.println(potVal); }

analogRead(potPin) will return the current value of our analog potentiometer, we store this in our variable potVal so that we can use it later in the code for conditions

Serial.println(potVal); will send the potentiometers value to the debugging console so we can easily look for problems if there are any

we will now program the conditions that will light up the leds, keep in mind that the potentiometer goes from values 0 to 4095:

void loop() { potVal = analogRead(potPin); Serial.println(potVal); if (potVal <= 1000) { digitalWrite(rLedPin, HIGH); } else { digitalWrite(rLedPin, LOW); } }

we start by making an if statement and inside the () we write the actual condition
in this case being:
if the value of the potentiometer is lesser than or equals to 1000
if those conditions are met we execute the code inside the {} brackets

in our case it is digitalWrite(pin, HIGH) which will send a message to the GPIO pin we specified asking for it to be at the high value
at the end of our } bracket we type else and start new brackets. this will execute code if the condition above was NOT met, in our case if isn't met we send a message to the GPIO setting it to LOW

Let's write the rest of the conditions for the rest of the LED's

void loop() { potVal = analogRead(potPin); Serial.println(potVal); if (potVal <= 1000) { digitalWrite(rLedPin, HIGH); } else { digitalWrite(rLedPin, LOW); } if (potVal <= 2000 && potVal >= 1001) { digitalWrite(yLedPin, HIGH); } else { digitalWrite(yLedPin, LOW); } if (potVal <= 3000 && potVal >= 2001) { digitalWrite(gLedPin, HIGH); } else { digitalWrite(gLedPin, LOW); } }

for the next if conditions we will need to check if the potentiometer value is lesser than AND higher than a value so that we can have a specific range of it being active, we declare and and statement inside of an if statement by typing && and then typing the other condition.
then we do the same thing for the else conditions as we did before

this should be all that we need to go you may now send the full code to the board and test it out!
here is the full code but I recommend you follow along with the steps above instead of just copying it

const int rLedPin = 18; const int yLedPin = 19; const int gLedPin = 21; const int potPin = 4; int potVal = 0; void setup() { Serial.begin(115200); // Starts Serial for debugging pinMode(rLedPin, OUTPUT); pinMode(yLedPin, OUTPUT); pinMode(gLedPin, OUTPUT); } void loop() { potVal = analogRead(potPin); Serial.println(potVal); if (potVal <= 1000) { digitalWrite(rLedPin, HIGH); } else { digitalWrite(rLedPin, LOW); } if (potVal <= 2000 && potVal >= 1001) { digitalWrite(yLedPin, HIGH); } else { digitalWrite(yLedPin, LOW); } if (potVal <= 3000 && potVal >= 2001) { digitalWrite(gLedPin, HIGH); } else { digitalWrite(gLedPin, LOW); } }
